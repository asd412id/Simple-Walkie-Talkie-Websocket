# Simple-Walkie-Talkie-Websocket
Simple Walkie-Talkie Application using Socket.io

## How to Use:
- Clone this repo
- Open terminal/cmd and point to the directory
- Run **npm install**
- Run **npm start**
- Open **https://localhost:3000** on your favorite browser
